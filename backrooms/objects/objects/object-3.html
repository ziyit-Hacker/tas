Object 3 - “驱笑剂”

Object 3: 驱笑剂
等级： 安全 物品稀有，在多数层级皆有发现。 对多数流浪者的生存至关重要。

描述：

驱笑剂从一株植物上滴落。

Object 3是一种淡蓝色荧光粘稠液体，通常被密封在刻有螺纹的铜制圆柱形容器中。静置时，液体会自主产生持续而细微的气泡，并在容器内壁留下短暂的磷光痕迹。当暴露在空气中，驱笑剂会以每小时5毫升的速度缓慢蒸发，形成带有甜杏仁气味的淡蓝色薄雾，这种薄雾会在重力作用下自然沉降，并在地表形成独特的结晶图案。

在 Level 500 的特定月光相位下，新合成的驱笑剂会呈现出类似星云的光学特性：在完全黑暗的环境中，液体会自主发出脉冲式的微光，其亮度与 Level 500 的月光强度呈正相关。失效样本虽保留物理特性，但不再发光，且蒸发速率提升至正常的三倍。

特殊物理性质研究显示，驱笑剂具有独特的量子叠加状态：当未被观测时，液体会同时存在于容器内的所有位置，这也是为何即使瓶罐破损，液体也从未发生泄漏。只有在有意识的观察者注视下，它才会坍缩为确定的液态形态。这一特性使其成为后室物理学研究的重要样本，M.E.G.研究部门已将其列为"理解后室时空异常性的关键介质"。

在 Level 500 的月光达到峰值时，新合成的驱笑剂会展现出更显著的异常特性。此时若将制剂倾倒在镜面表面，液体会自发组织成复杂的几何图案，这些图案被证实与失落一族的古老符文有着惊人的相似性。有理论认为，驱笑剂实际上是液态化的符文能量，这一发现或许能解释为何它能够影响后室基础物理规律。

特性：

符文依赖性合成

  • 仅能通过激活状态的符文 A - EPE-500-SRH 在纯铜容器中完成合成，合成过程需持续暴露在 Level 500 的直射月光下。

实体干扰场

  • 挥发的薄雾可在半径3米内形成感知干扰场，使笑魇等实体暂时失去对人类的追踪能力，效果持续约15分钟。

危险化学反应

  • 与腰果水接触会引发剧烈爆炸（当量相当于200克TNT）。
  • 与杏仁水混合将立即中和为普通蒸馏水
  • 与火盐接触会产生持续性紫色火焰。

相位敏感性

  • 这在新月期间合成的制剂完全无效，满月期间合成的制剂效果增强50%。

事件：

  • 事件A（2009.9.13）：因符文能量周期性衰减，全球现存驱笑剂在2小时内同时失效，后续实验证实失效样本与腰果水仍保持爆炸反应
  • 月光重构事件（2025.10）：失落一族发现月光相位与符文活性的量子纠缠关系，通过校准符文石刻的角度成功恢复合成协议
  • 混合实验事故：M.E.G.研究员试图将驱笑剂与液态痛苦混合，导致整个实验室被转化为 Level 37 的临时入口

行为准则：

应当：

  • 使用纯度≥99%的铜制容器储存，每次使用后立即密封。
  • 合成前必须确认 Level 500 处于可见月光照射下。
  • 在遭遇笑魇群时，将制剂滴洒在周围形成防护圈。

不应：

  • 不可同时携带驱笑剂与腰果水。
  • 使用后24小时内避免接触任何形式的杏仁水。
  • 禁止在 Level 500 的阴影区域进行合成尝试。
  • 饮用瓶子半空的杏仁水。

报告：

  • 发现失效样本仍具挥发性时立即向M.E.G.化学危害部门报备。
  • 观测到驱笑剂异常发光现象时记录当前月光相位。
